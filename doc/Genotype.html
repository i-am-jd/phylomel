Genotype functions

This module implements a genotype and its common functions

type t = {
   id : string;
   markers : float array;
   infos : string array; (*exemple : |France; Paris; 2009|*)
} 
val size : t -> int
size g
Returns number of markers
val diff : t -> t -> int
diff g1 g2
Returns number of different markers
val markers_nb : t array -> int option
markers_nb gs
Returns - Some size where size if the common number of markers of the genotypes
  • None if the genotypes have different sizes

val description : t -> string
description g
Returns the concatenation of g.id and g.infos
